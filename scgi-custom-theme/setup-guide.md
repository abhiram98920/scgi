# SCGI Custom WordPress Theme - Setup Guide

This theme converts the static SCGI HTML site into a fully dynamic WordPress experience with completely automated setup.

## 1. Zero-Setup Installation

Unlike traditional themes, this custom SCGI theme contains an auto-population script. When you activate the theme, it automatically builds the entire site to look exactly like the HTML design.

1. Upload the `scgi-custom-theme.zip` file (or the unzipped folder) to your WordPress site under **Appearance > Themes > Add New > Upload Theme** (or via FTP to `/wp-content/themes/`).
2. Go to **Appearance > Themes** and click **Activate** on the "SCGI Custom Theme".

### What Happens Automatically When You Click Activate?
- **Pages Created:** Home, About Us, Courses, Gallery, Contact.
- **Home Page Set:** The "Home" page is automatically assigned as the static front page.
- **Navigation Setup:** The primary menu is created and all pages are assigned to it automatically.
- **Contact Info Pre-filled:** Phone numbers, addresses, and email are automatically inserted into the Customizer.
- **Data Imported:** All Sliders, Courses, Logos, and Gallery Images are automatically created with their matching text and images downloaded to your Media Library.

**You do NOT need to import any SQL files.** Simply activate the theme, and your site is ready.

## 2. Managing Content

Everything generated automatically is fully editable via the WordPress Dashboard Sidebar.

### Sliders (Home Hero)
- Go to **Sliders > All Sliders**.
- You can edit the prominent Golden Subtitle, the short paragraph description, and button links.
- Uses the **Featured Image** for the background.

### Courses
- Go to **Courses > All Courses**.
- Details like duration, excerpts, and category badges are easily modified.

### Accreditations (Logos)
- Go to **Logos > All Logos**.
- To add a new accreditation logo to the scrolling marquee, just Add New and set a Featured Image.

### Customizer
- Go to **Appearance > Customize > Contact Information**.
- This is where you centrally update the Phone Number, WhatsApp, Email, and Map Address location. Changes apply everywhere instantly.
